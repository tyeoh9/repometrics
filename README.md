# repometrics

A CLI that measures repository health.

---

## Install

```bash
pip install repometrics
```

---

## Usage

```bash
repometrics scan
repometrics scan --days 60
repometrics scan --json
```

---

## Example

```text
Repo Health Score: 74 / 100

Structure:     68
Dependencies:  80
Git Activity:  72
Hygiene:       75
```
